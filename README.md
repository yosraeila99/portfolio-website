# Personal Portfolio Website

A modern responsive personal portfolio website built with React and Vite.  
The project presents a clean dark UI design with a hero section, Arabic and English content, skills, selected projects, social links, and a working contact form using Formspree.

## Project Overview

This portfolio website was built to showcase personal skills, projects, and contact information in a professional and modern way.

The website focuses on combining clean visual design with clear Arabic and English content, making it suitable for personal branding, freelance profiles, and frontend portfolio presentation.

## Features

- Modern dark user interface
- Responsive design for desktop and mobile
- Hero section with social links
- Arabic and English introduction content
- Skills section with icons
- Projects section with live demo and source code buttons
- Working contact form using Formspree
- Smooth section navigation
- Clean and organized React structure

## Sections

- Hero Section
- About Me
- Skills

- Contact Form
- Footer

## Technologies Used

- React
- Vite
- JavaScript
- CSS
- React Icons
- Formspree

## Contact Form

The contact form is connected using Formspree, allowing visitors to send messages directly without needing a custom backend.
